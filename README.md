# MCP Server Demo

A demonstration project for MCP server functionality.

## Description

This repository contains a demo server project initialized with npm. It serves as a starting point for developing server applications.

## Installation

```bash
git clone https://github.com/rpinedaMC/mcp-server-demo.git
cd mcp-server-demo
npm install
```
